## About The Project
    

     Create a function that takes a string and returns the number (count) of vowels contained within it.
     
      Examples
      countVowels("Celebration") ➞ 5
      countVowels("Palm") ➞ 1
      countVowels("Prediction") ➞ 4
     
     Exercices are from: https://edabit.com/challenges/swift
